Maquina de Turing

primeira ideia de computador criado, por Alan Turing, inicio do seculo 20, 1930
4 elementos 
- 1 fita infinita dividade em celulas 
- 2 cabeçote capaz de ler a fita
- 3 registrador
- 4 função de transição

primeira ideia do algoritmo

definição de algoritmo (Conjunto finito de passos capaz de resolver um dado problema)

binario é a forma de comunicação e informação dos computadores, 1 bit é a unidade minima
onde 0 = desligado/ falso  1 = ligado/ verdadeiro

------

priemira geração de computadores  = Valvulas termionicas

segunda geração de comp. = transistores

terceira geração = circuito impressos

quarta geração = micro processadores

Módulo 2 

Hardware e Software

Hardware = componentes fisicos
 
Software = programas executados no computador

---

Hardware

*processador  = responsavel por executar as instruções no precederam
ele depende de um pulso periodico, chamado clock.
memoria cache no processador, é uma pequena memoria que melhora
a performance

*Memoria principal ou RAM = responsavel pro guardar os dados e processador
memoria de acessorio aleatorio, onde você sempre usa ela e
para processar outra, você precisa descartar a anterior.
Quanto maior a quantidade de MHz mais rapido vai ser o processamento.

* Placa Mãe = circuito eletronico impresso, onde vai ser
encaixado os outros componetes para conectar as peças principais
e perifericos, para interagirem e o computador funcionar.
Você coloca as peças nos chamados barramentos, é colocado
processadores, memorias, hd e perifericos.

* perifericos = 
dispositivos de entrada = mouse, teclado 
dispositivos de saida = monitor, impressora
dispositivos de entrada e saida = monitor touchscreen


Software

Software Finalísticos ou de aplicação = Navegadore de internet,
planilhas, editores de texto, jogos eletronico e etc.

Software de Sistemas = Driver de dispositivos
